Strategy Backtest

Ranks your watchlist by trailing momentum at each rebalance, holds the top N equal-weighted, and compares the result to a buy-and-hold benchmark of the same list.

Backtest Settings

Uses the same watchlist as the screener above. Each ticker costs 1 API call (historical price series). Transaction cost and volume-based slippage are now modeled. Validation periods split your date range into independent, non-overlapping chunks so one lucky stretch can't carry the whole result. Survivorship bias is still not corrected — your watchlist reflects stocks that exist today, not the full set that existed historically, and fixing that requires point-in-time constituent data this app doesn't have access to. Treat all results as directional, not a basis for real capital.
